# Focus Shield Browser Extension -- Store Listing

## Name

Focus Shield -- Distraction Blocker

## Short Description (132 chars max for Chrome)

Block distracting websites during focus sessions. Configurable blocklists, motivational quotes, breathing exercises.

## Description

Focus Shield helps you stay productive by blocking distracting websites during focus sessions.

**Features:**

- Block social media, entertainment, news, and shopping sites
- Custom blocklist with wildcard patterns (e.g., `*.reddit.com`)
- Beautiful blocked page with motivational quotes
- Built-in breathing exercise for mindfulness
- Session timer showing time remaining
- Distraction attempt counter
- Works with the Focus Shield desktop app for system-level blocking

This extension works standalone or paired with the Focus Shield desktop app for enhanced blocking capabilities (hosts file modification, process blocking).

Focus Shield is **100 % open-source** and **free** -- no subscriptions, no ads, no tracking.

## Privacy policy

Focus Shield does not collect, transmit, or store any personal data. All data remains on your device. The extension communicates only with the local Focus Shield desktop application (localhost) and never with external servers unless you explicitly configure a self-hosted sync server.
