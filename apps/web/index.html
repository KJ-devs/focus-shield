Everything You Need to Stay Focused

A comprehensive suite of tools designed to help you build better work habits and eliminate distractions.

Session Engine

Pomodoro, Deep Work, Sprint, and fully custom session sequences. Scheduling, auto-start, and smart pause when your PC locks.

Smart Locking

Five friction levels from gentle to nuclear. Cryptographic token system with Argon2 hashing, cooldowns, and anti-paste protection.

Multi-Level Blocking

Browser extension, DNS/hosts file modification, and process management. Three layers working together to stop distractions.

Analytics Dashboard

Heatmaps, trend charts, radar diagrams, and focus scores. Track your productivity over days, weeks, and months.

Gamification

Streaks, XP, levels, and achievements keep you motivated. Unlock cosmetic themes as you build your focus habit.

Self-Hosted Sync

Optional NestJS + PostgreSQL sync server you can run on your own hardware. Multi-device sync, buddy system, and challenges.

How It Works

Three simple steps to reclaim your attention.

1

Configure

Choose a preset (Pomodoro, Deep Work, Sprint) or build your own custom session. Select which sites and apps to block, and set your desired lock level.

2

Focus

Start your session and get to work. Distracting websites are blocked in your browser and at the system level. The cryptographic lock keeps you committed.

3

Review

When your session ends, review your focus score, see how many distractions were blocked, and track your progress over time in the analytics dashboard.

Run Your Own Sync Server

Keep your data entirely under your control. Deploy the sync server with Docker in under a minute.

terminal 
# Clone the repository
git clone https://github.com/KJ-devs/focus-shield.git
cd focus-shield/apps/sync-server

# Start everything with Docker Compose
docker compose up -d

# The sync server is now running on http://localhost:3001

Includes NestJS API server and PostgreSQL database. See the self-hosting guide for detailed configuration options.
